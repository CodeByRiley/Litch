# Litch

First attempt at making a game and trying to publish.
Meant to be an Action-RPG / Roguelike.

## Features:
- [x] Basic Scene Manager
- [x] Raylib Inputs
- [ ] Basic Asset Manager
- [ ] Basic Audio Manager
- [x] Basic UI Manager
